cb 与 队列插件 结合使用

cb 一般写在异步回调函数内。

关于 cb 自己的理解：就像 JAVA 多线程处理同步的锁一样，只要 task 的回调函数有 cb，这个任务就会在内存中执行，其他任务不会执行，直到 cb( )，才会释放锁，其他的 task 才会继续执行。
